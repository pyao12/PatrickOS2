# 欢迎来到 Patrick Operating System 2！

上一个版本的 PatrickOS 存在结构混乱、漏洞百出的问题，因此有了 Patrick OS 2 这个重构版本！

整个代码库都完全重构，为您带来焕然一新的体验！

## 编译

首先需要安装必要的库：

```bash
sudo apt update
sudo apt install make qemu-system-x86 clang lld 
sudo apt install dosfstools mtools clang-format grub-common grub-pc
```
